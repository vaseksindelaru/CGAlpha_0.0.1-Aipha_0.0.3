# .learning/ - Plan de Aprendizaje Python + CGAlpha

**Propósito**: Seguimiento del progreso de aprendizaje de Python mientras se construye CGAlpha v2.

## Contenido

| Archivo/Directorio | Propósito |
|-------------------|-----------|
| `LEARNING_PLAN.md` | Plan completo de aprendizaje |
| `learning_progress.json` | Progreso actual (machine-readable) |
| `daily_logs/` | Logs diarios de aprendizaje |
| `questions/` | Preguntas hechas a agentes IA |
| `exercises/` | Ejercicios completados/pendientes |

## Uso

Este directorio es para **tu aprendizaje personal**. Contiene:
- Tu plan de aprendizaje
- Tu progreso día a día
- Tus preguntas y respuestas
- Tus ejercicios

## Cualquier LLM puede continuar

```bash
# Prompt para nuevo LLM:
"Lee .learning/learning_progress.json y .learning/daily_logs/daily_log.jsonl.
Continúa la enseñanza desde donde se quedó.
Sigue el formato establecido en LEARNING_PLAN.md."
```
