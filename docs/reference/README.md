# Reference Docs (CGAlpha)

Este directorio contiene referencias operativas extraídas del documento monolítico de constitución.

Objetivo:
- navegación rápida,
- menor riesgo de desincronización,
- uso directo por operador y por LLM local.

## Archivos

- `docs/reference/constitution_core.md`: reglas núcleo y límites no negociables.
- `docs/reference/gates.md`: gates de readiness para Deep Causal / v0.3.
- `docs/reference/parameters.md`: parámetros críticos y rangos operativos.

## Regla de mantenimiento

Cuando cambie una política:
1. actualizar primero este `docs/reference/`,
2. luego reflejar en `UNIFIED_CONSTITUTION_v0.0.3.md` si aplica histórico/legal.
