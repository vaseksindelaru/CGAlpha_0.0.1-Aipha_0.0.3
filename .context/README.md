# .context/ - Contexto de Reconstrucción CGAlpha v2

**Propósito**: Contexto persistente para LLMs que trabajan en la reconstrucción técnica de CGAlpha.

## Contenido

| Archivo | Propósito |
|---------|-----------|
| `CHANGELOG.md` | Historial de cambios de la reconstrucción |
| `migration_status.json` | Estado de migración (machine-readable) |
| `decisions.jsonl` | Architecture Decision Records (ADRs) |
| `prompts/` | Prompts para cada fase de reconstrucción |
| `PHASE_1_ANALYSIS.md` | Análisis inicial del LLM avanzado |
| `ACTION_REPORT.md` | Informe de acciones de reestructuración |

## Uso

Este directorio es para **LLMs que reconstruyen CGAlpha**. Contiene:
- Decisiones arquitectónicas
- Estado de migración
- Prompts para continuar el trabajo
